A browser-based tool for building, posing, and exporting **Character Bible JSON files** for use with AI image generators (Midjourney, Leonardo AI, Stable Diffusion, and more).

---

## 🚀 What It Does

The 3C Character Pose Builder solves the **#1 problem in AI character generation** — consistency.

Instead of re-describing your character every time and getting a different result, you build a **complete character reference package** (skeleton pose + visual bible) and export it as a single JSON file. Feed that JSON into any AI workflow and your character stays *your* character — same face, same clothes, same identity, every time.

---

## ✨ Features

### 🦴 Pose Canvas
- **16-joint interactive skeleton** — drag joints to any position
- **Quick pose presets** — Stand, Arms Out, Thumbs Up, Running, Sitting, Flying, Pointing
- **Mirror pose** — flip horizontally with one click
- **Export skeleton PNG** — save the pose as a transparent PNG

### 📖 Character Bible
Upload reference images and add descriptors for:

| Feature | Description |
|---|---|
| 💇 Hair | Style, colour, texture |
| 👁 Eyes | Shape, colour, lash style |
| 👃 Nose | Shape, size |
| 💋 Lips | Style, expression, colour |
| 🎨 Skin Tone | Colour picker + descriptor |
| 🤲 Hands | Style, details |
| 👕 Outfit Top | Shirt, suit, armour |
| 👖 Outfit Bottom | Shorts, trousers |
| 👟 Shoes / Boots | Style, colour |
| 🦸 Cape / Extra Layer | Capes, wings, coats |
| 💎 Accessories | Badges, wristbands, effects |

### ⚡ Auto-Generated AI Prompt
Builds a generic, ready-to-paste prompt from all your character details — compatible with Midjourney, Leonardo AI, and any other generator.

### 💾 Export / Import
- Export a complete **self-contained JSON file** to your computer
- Load any previously saved JSON back in to continue editing
- Base64 images embedded — **no server needed, no internet required**

## 🏗 How To Use

### Option 1 — GitHub Pages
1. Fork this repo
2. Go to **Settings → Pages → Deploy from branch → main → / (root)**
3. Your tool is live at `https://[your-username].github.io/3c-character-pose-builder`

### Option 2 — Local
1. Download `index.html`
2. Open in any browser — no install, no server needed

---

## 🔁 Workflow

```
1. Open the app
2. Name your character + give the pose a title
3. Drag joints to set the body position
4. Upload reference images + add descriptors in the Character Bible
5. Export JSON → save to your computer
6. Paste the auto-generated AI prompt into your generator
7. Load your JSON any time to continue editing
```
