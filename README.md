## ЁЯЪА What It Does

The 3C Character Pose Builder solves the **#1 problem in AI character generation** тАФ consistency.

Instead of re-describing your character every time and getting a different result, you build a **complete character reference package** (skeleton pose + visual bible) and export it as a single JSON file. Feed that JSON into any AI workflow and your character stays *your* character тАФ same face, same clothes, same identity, every time.

## ЁЯФБ Workflow

```
1. Open the app
2. Name your character + give the pose a title
```
